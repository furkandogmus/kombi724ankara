# Görseller / Resimler

Bu klasöre **gerçek resimlerinizi** aşağıdaki dosya adlarıyla ekleyin.
Bir resim eksikse site otomatik olarak yer tutucu (placeholder) gösterir,
yani site yine de çalışır.

## Gerekli dosyalar

| Dosya adı          | Nerede kullanılır            | Önerilen boyut       |
|--------------------|------------------------------|----------------------|
| `hero-kombi.jpg`   | Ana sayfa üst görsel (hero)  | 1200 × 900 px        |
| `galeri-1.jpg`     | Galeri – Kombi bakımı        | 800 × 600 px         |
| `galeri-2.jpg`     | Galeri – Petek temizliği     | 800 × 600 px         |
| `galeri-3.jpg`     | Galeri – Arıza tespiti       | 800 × 600 px         |
| `galeri-4.jpg`     | Galeri – Parça değişimi      | 800 × 600 px         |
| `galeri-5.jpg`     | Galeri – Kombi revizyonu     | 800 × 600 px         |
| `galeri-6.jpg`     | Galeri – 7/24 acil servis    | 800 × 600 px         |
| `og-kapak.jpg`     | Sosyal medya paylaşım kapağı | 1200 × 630 px        |
| `favicon.png`      | Tarayıcı sekme ikonu         | 64 × 64 px           |

## İpuçları
- Dosya adlarını **birebir** aynı tutun (küçük harf, Türkçe karakter yok).
- Telefonla çektiğiniz fotoğrafları küçültmek için https://squoosh.app kullanabilirsiniz (yükleme hızını artırır).
- `.jpg` yerine `.webp` kullanmak isterseniz `index.html` içindeki `src` yollarını da güncelleyin.
